### To restore database from an dump file 
```bash
psql -d employee_journal_dev -h localhost -U postgres < '/home/kushal/work-diary-dev-21-mar-2023-backup.dump'
```
- `-d` : Database name
- `-h` : Host name
- `-U` : Username (Keep Default : **postgres**)
